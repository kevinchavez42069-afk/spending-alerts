Connect GitHub

To save your categorization choices so future spending checks pick them up automatically, this page needs a GitHub token that can edit categories.json in this repo.

  1. Go to github.com/settings/personal-access-tokens/new
  2. Under Repository access, choose Only select repositories → spending-alerts
  3. Under Permissions → Repository permissions, set Contents to Read and write
  4. Generate the token and paste it below

The token is stored only in this browser's local storage — it's never sent anywhere except directly to GitHub's API, and never committed to the repo.
